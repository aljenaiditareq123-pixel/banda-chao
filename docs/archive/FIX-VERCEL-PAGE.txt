🎯 الطريقة 1: استخدم Vercel CLI (أسهل!)

   افتح Terminal واكتب:

🎯 الطريقة 2: ارفع على GitHub أولاً (5 دقائق)

   1. اذهب: https://github.com/new
   2. اسم المستودع: banda-chao
   3. اضغط "Create repository"
   4. انسخ الرابط
   5. نفّذ في Terminal:
      
      cd /Users/tarqahmdaljnydy/Desktop/banda-chao
      git remote add origin [الرابط]
      git push -u origin main
   
   6. ثم ارجع لصفحة Vercel واختر المستودع

   - ابحث عن زر "Deploy" أو "Upload"
   - أو اضغط "..." أو "More options"
   - ابحث عن "Upload" أو "Local folder"
